Groovy.
This article gives a short overview of the Groovy language
including collections, loops, gstrings, MOP, closures, operator
overloading, XML
handing and using Groovy together with Java class.
It also
describes how to
use the
Eclipse IDE for developing Groovy.
This article assumes that you have already Eclipse installed
and that you have used Eclipse for Java development. This article
was written using Groovy 2.4, Eclipse 4.4 (Luna) and Java 1.8.
